# FundDonate - Setup Instructions

## Installation Steps

1. Clone Repository
git clone https://github.com/Amruta-32/FundDonate
cd FundDonate

2. Install Dependencies
pip install -r requirements.txt

3. Database Setup
Start XAMPP and run MySQL
Create database: fund_donation_db

4. Run Application
python app.py

5. Access Application
Open: http://localhost:5000


Database Config

Host: localhost

Database: fund_donation_db

Username: root

Password: (none)
